# Calendar Management API

## Accès à la documentation Swagger

La documentation interactive de l'API est disponible à l'adresse suivante après avoir lancé le backend :

```
http://localhost:3000/api-docs
```

Vous pouvez y consulter et tester tous les endpoints disponibles.

## Sécurisation de l'API

L'API utilise les middlewares Helmet et CORS
